Understanding Audiometric Results

Audiogram Interpretation

Learn how to interpret audiometric results and identify different types of hearing loss.

Key Components:

  • Degree of hearing loss (mild to profound)
  • Type of hearing loss (conductive, sensorineural, mixed)
  • Configuration of hearing loss

Tympanogram Results

Types of Tympanograms:

  • Type A - Normal middle ear function
  • Type B - Flat tympanogram
  • Type C - Negative middle ear pressure
  • Type Ad - Hypermobile tympanic membrane
  • Type As - Stiff middle ear system
